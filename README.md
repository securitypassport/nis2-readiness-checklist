# NIS2 Readiness Checklist

A practical checklist to help organizations prepare for NIS2 compliance.

## Governance

* Executive accountability defined
* Cybersecurity responsibilities assigned
* Risk management process documented

## Asset Management

* Asset inventory maintained
* Critical systems identified
* Third-party dependencies documented

## Access Control

* Multi-factor authentication enforced
* Privileged access reviewed
* User lifecycle process documented

## Incident Management

* Incident response plan documented
* Escalation process defined
* Reporting obligations understood

## Business Continuity

* Backup strategy documented
* Recovery procedures tested
* Continuity responsibilities assigned

## Supply Chain Security

* Vendor assessments performed
* Third-party risk reviewed
* Security requirements documented

## Security Monitoring

* Logging enabled
* Monitoring procedures established
* Vulnerability management process defined
